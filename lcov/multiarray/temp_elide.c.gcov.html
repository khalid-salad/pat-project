      17             :  *        0 LOAD_FAST                0 (a)
      18             :  *        3 LOAD_FAST                1 (b)

      21             :  *       10 BINARY_ADD
      22             :  *
      23             :  * The two named variables get their reference count increased by the load

      25             :  * The temporary of the first BINARY_ADD on the other hand only has a count of
